## CARA INSTALL DI TERMUX
```bash
> pkg install nodejs && pkg install git
> git clone https://github.com/frostkri/xuzynri-md
> cd xuzynri-md
> bash install.sh
> npm start/node main.js
```
## CARA INSTALL DI LAPTOP
```bash
> git clone https://github.com/forstkri/xuzynri-md
> cd xuzynri-md
> npm i
> npm start/node main.js
```
